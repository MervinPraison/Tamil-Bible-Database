Part 5 - Cheat sheet

This section contains summary of symbols discussed earlier

Description Code
Specify Breakpoints in title or in verse {br}
Specify title location within a verse {t}
Specify hierarchy of header within a verse {t}
Specify Indents {i} and {/i}
Specify Outdents {o} and {/o}
Poem style 1 {q1} and {/q1}
Poem style 2 {q2} and {/q2}
Poem style 3 Specifying the above poem style 1 without a new line
Sample verse format ~
v/1
தொடக்கத்தில் கடவுள் விண்ணுலகையும், மண்ணுலகையும் படைத்தபொழுது,
~
Coupled verse format ~
v/7-8
அதற்கு எஸ்தர், “என் விண்ணப்பமும் ...
~
v/8
Same as above
~
Header format (Verse number here after which header occurs) ~
t/0
கடவுள் உலகைப் படைத்தல்
~
Footnotes and Cross references ~
c/3 2 கொரி 4:6
f/27 ‘ஆதாம்’ என்பது எபிரேய பாடம்
~
